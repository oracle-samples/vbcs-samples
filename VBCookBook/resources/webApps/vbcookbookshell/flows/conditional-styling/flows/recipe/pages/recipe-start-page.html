Add a Table Component

  1. In the Component Palette, drag a Table component and drop it on the page.
  2. Use the Table component's Add Data Quick Start to add data to your table.

Add a LED Gauge Component

  1. In the Component Palette, drag a LED Gauge component and drop it on the Rating column of the Table component.
  2. In the Data tab of the Property Inspector for the LED Gauge component, set the following property values:
    • Value: [[$current.data]]
    • Threshold: [{"max": 2}, {"max": 4}, {}]

    This adjusts the background color based on the value in the Rating column.
