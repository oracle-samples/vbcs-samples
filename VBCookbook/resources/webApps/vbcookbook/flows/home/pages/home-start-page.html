Browse through the samples using the navigation pane to view implementations of common usage patterns developed using Visual Builder. Each sample displays a page with the implementation, plus some high-level steps that describe how to implement the sample using Visual Builder.

If you want to examine the source code to gain a better understanding of the sample, Oracle publishes the Visual Builder Cookbook as an application template, so all you need to do is create an application in your instance of Visual Builder where you select Oracle Visual Builder Cookbook as the application template. View the step-by-step instructions plus the specific location of each sample within the source by clicking Open in Visual Builder.
