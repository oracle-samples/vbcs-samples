This is the source project for the ApplicationsCloudUI sample theme for JET. You can test out the theme by adding various components to the page below.

Note: If you just see the normal Oracle JET look and feel make sure that you run the project with the theme flag e.g.

ojet serve --theme=ApplicationsCloudUI:web

Once you have built or run this project you will find a deploy/ApplicationsCloudUI-export.zip containing the generated theme files in the /deploy folder. You can take this file and import it within the /resources/css folder of your VBCS application
